# 세얼환경 웹사이트

정적 HTML 사이트입니다. (index.html)

## 배포 (GitHub + Vercel)
1. 이 폴더의 파일을 GitHub 저장소에 업로드
2. Vercel에서 해당 저장소를 Import
3. Framework Preset: Other / Build Command: 없음 / Output: 루트
4. Deploy 클릭 -> *.vercel.app 주소 생성

## 사진 추가
- images 폴더에 사진(영문 파일명)을 넣고 index.html에서 <img src="images/파일명.jpg"> 로 사용
